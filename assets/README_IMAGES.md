# הוראות להוספת תמונות

אנא שמור את התמונות הבאות בתיקייה זו (`assets`) עם השמות הבאים:

## קבצים נדרשים:

### 1. לוגו החברה
- **שם קובץ:** `logo.png`
- **תיאור:** הלוגו של המדביק בע״מ (הלוגו עם החץ והגלגל)

### 2. תמונת המייסד
- **שם קובץ:** `founder.jpg`
- **תיאור:** תמונת משה אליקים ז״ל, המייסד

### 3. תמונת רחמים הצעיר
- **שם קובץ:** `rahamim_young.jpg`
- **תיאור:** רחמים אליקים הצעיר עובד במפעל עם אביו משה

### 4. תמונות גלריה
- **שם קובץ:** `gallery_1.jpg`
- **תיאור:** רחמים אליקים הצעיר עובד במפעל עם אביו משה

- **שם קובץ:** `gallery_2.jpg`
- **תיאור:** עובדים במפעל המדביק בעבודה יומיומית (מבט רחב על המפעל)

- **שם קובץ:** `gallery_3.jpg`
- **תיאור:** רחמים אליקים עובד במפעל (עם כובע ירוק)

- **שם קובץ:** `gallery_4.jpg`
- **תיאור:** עובדי המדביק בחדר האוכל

- **שם קובץ:** `gallery_5.jpg`
- **תיאור:** צוות המדביק - תמונה קבוצתית (15 אנשים)

- **שם קובץ:** `gallery_6.jpg` (אופציונלי)
- **תיאור:** מבט כללי על המפעל

## הוראות:

1. שמור את כל התמונות בתיקייה `assets`
2. ודא שהשמות מדויקים (כולל סיומות הקבצים: .jpg, .png)
3. התמונות יוצגו אוטומטית באתר לאחר שתשמור אותן

## הערות:

- התמונות יכולות להיות בפורמטים: JPG, PNG, JPEG
- מומלץ שהתמונות יהיו באיכות טובה (אבל לא גדולות מדי - עד 2MB כל אחת)
- האתר יתאים את התמונות אוטומטית לגודל המתאים
