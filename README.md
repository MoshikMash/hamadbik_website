# אתר המדביק בע״מ

אתר אינטרנט לחברת המדביק בע״מ - חידוש חלקי חילוף לרכב

## קבצים נדרשים

יש להציב את הקבצים הבאים בתיקיית `assets`:

1. **logo.png** - הלוגו של החברה (חסר - יש להוסיף)
2. **founder.png** - תמונת משה אליקים ז״ל, המייסד ✓
3. **rahamim_young.png** - תמונת רחמים אליקים הצעיר עובד במפעל עם אביו ✓
4. **gallery_1.png** - רחמים אליקים הצעיר עובד במפעל עם אביו משה ✓
5. **gallery_2.png** - עובדים במפעל המדביק בעבודה יומיומית ✓
6. **gallery_3.png** - רחמים אליקים עובד במפעל ✓
7. **gallery_4.png** - עובדי המדביק בחדר האוכל ✓
8. **gallery_5.png** - צוות המדביק - תמונה קבוצתית ✓

## הפעלת האתר

פשוט פתח את הקובץ `index.html` בדפדפן.

או הפעל שרת מקומי:
- Python: `python -m http.server 8000`
- Node.js: `npx http-server`

## תכונות

- עיצוב מודרני ורספונסיבי
- תמיכה בעברית (RTL)
- מוכן לתרגום לאנגלית (כפתור שפה מוכן לעתיד)
- סעיף "הסיפור שלנו" עם מורשת המייסד
- מידע מלא על השירותים והתקשרות
