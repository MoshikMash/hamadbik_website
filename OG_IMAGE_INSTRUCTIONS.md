# הוראות ליצירת תמונת Open Graph ל-WhatsApp

## דרישות התמונה:
- **גודל:** 1200x1200 פיקסלים (ריבועי - יחס 1:1)
- **פורמט:** PNG או JPG
- **שם הקובץ:** `og-image.png`
- **מיקום:** בתיקיית `assets/`

## למה זה חשוב?
כשמישהו שולח את הקישור לאתר ב-WhatsApp, WhatsApp מציג תצוגה מקדימה עם תמונה, כותרת ותיאור. התמונה צריכה להיות בפרופורציות הנכונות כדי להיראות מושלם.

## ⚠️ חשוב: איך למנוע חיתוך הלוגו

**הבעיה:** WhatsApp עלול לחתוך את התמונה אם הלוגו קרוב מדי לקצוות.

**הפתרון:** הוסף **רווחים (padding)** סביב הלוגו!

### טיפים למניעת חיתוך:
1. **אל תשים את הלוגו בקצה** - השאר לפחות 100-150 פיקסלים מכל צד
2. **ממורכז את הלוגו** - שים אותו במרכז התמונה
3. **השתמש ב"Safe Zone"** - שמור את התוכן החשוב במרכז 800x800 פיקסלים
4. **הוסף רקע** - רקע צבעוני או לבן מסביב ללוגו

### מבנה מומלץ:
```
┌─────────────────────────┐
│                         │  ← רווח 150px
│    ┌─────────────┐      │
│    │             │      │
│    │    לוגו     │      │  ← במרכז
│    │             │      │
│    └─────────────┘      │
│                         │  ← רווח 150px
└─────────────────────────┘
     1200x1200px
```

## איך ליצור את התמונה:

### אפשרות 1: Canva (הכי קל - מומלץ)
1. לך ל-https://www.canva.com
2. לחץ על "צור עיצוב" → "עיצוב מותאם אישית"
3. הגדר גודל: **רוחב: 1200px, גובה: 1200px** (ריבועי)
4. **הוסף רקע** (גרדיאנט כחול או לבן)
5. **הוסף את הלוגו** (`assets/logo.png`) - גרור למרכז
6. **ודא שיש רווח** של לפחות 150px מכל צד של הלוגו
7. הוסף טקסט (אופציונלי):
   - כותרת: "המדביק בע״מ" (גדול ובולט)
   - תת-כותרת: "חידוש חלקי חילוף לרכב מאז 1968"
8. לחץ על "הורד" → בחר "PNG" → שמור
9. שמור את הקובץ כ-`og-image.png` בתיקיית `assets/`

### אפשרות 2: PowerPoint / Office
1. צור מצגת חדשה
2. שנה את גודל השקופית: **1200x1200 פיקסלים**
   - Design → Slide Size → Custom → 1200x1200px
3. הוסף רקע (Format Background → Gradient Fill)
4. הוסף את הלוגו (Insert → Pictures)
5. **ממורכז את הלוגו** - ודא שיש רווח מכל צד
6. הוסף טקסט אם צריך
7. שמור כ-PNG: File → Save As → PNG
8. שמור כ-`og-image.png` בתיקיית `assets/`

### אפשרות 3: Photoshop / GIMP
1. צור מסמך חדש: **1200x1200 פיקסלים** (ריבועי)
2. הוסף רקע
3. הוסף את הלוגו במרכז
4. **ודא שיש Guides** ב-150px מכל צד (Safe Zone)
5. שמור כ-PNG
6. העלה ל-`assets/og-image.png`

## טיפים נוספים:
- ✅ הלוגו צריך להיות **ממורכז** במדויק
- ✅ השאר **רווחים גדולים** מסביב (150-200px)
- ✅ השתמש ב**רקע צבעוני** (לא שקוף) כדי שהלוגו יבלוט
- ✅ אם יש טקסט, ודא שהוא לא קרוב לקצוות
- ❌ אל תשים את הלוגו בקצה התמונה
- ❌ אל תמלא את כל התמונה בלוגו
